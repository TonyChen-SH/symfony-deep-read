

处理类: Kernel
请求类: Request
响应类: Response

```php
// public/index.php

// 1.创建一个内核类
$kernel = new Kernel($env, $debug);

// 2.用静态方法生成一个Request对象
$request = Request::createFromGlobals();

// 3.根据Request对象,获取Response对象
$response = $kernel->handle($request);

// 4.发送HTTP headers 和 content.
$response->send();

// 5.内核终止
$kernel->terminate($request, $response);
```
